JavaScript powstał w 1995 roku.
Jest to język skryptowy (interpretowany), nie musi zostać skompilowany do kodu maszynowego, aby zobaczyć jego efekty działania.
JavaScript jest wykonywany po stronie klienta, dzięki temu odciąża pracę serwerów.
Jest bezpiecznym narzędziem, ponieważ ma ograniczone uprawnienia dostępu do zasobów komputera.
Jest to język obiektowy, w języku JavaScript dokonuje się powiązania funkcji programu z danymi definiującymi przedmiot.

<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
tworzy guzik, który pozwala na otworzenie nowego okna

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
mówi o tym, jakie okno zostanie otworzone, jakie będzie miało wymiary i położenie

toolbar=no → pokazuje lub ukrywa standardowy pasek narzędzi
directories=no → pokazuje lub ukrywa przyciski katalogów
menubar=no → pokazuje lub ukrywa menu przeglądarki
height=280 → ustawia wysokość okna
width=160 → ustawia szerokość okna
top=200 → ustawia położenie okna względem góry ekranu
left=200 → ustawia położenie okna względem lewej strony ekranu

window.close()
służy do zamknięcia aktualnego okna
